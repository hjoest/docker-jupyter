* scikit-image
* Matplotlib
* SymPy
* Pandas
* NLTK

* Keras
* RISE
* Thrift

### Usage

      docker run -d --name jupyter -p 8888:8888 -v `pwd`:/notebook hjoest/jupyter
      docker logs jupyter

Open the displayed URL in a browser.
